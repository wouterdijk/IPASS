Detailed Description

mpu6050 accelerometer library

This is a library for the accelerometer within the mpu6050. In this library you can wake the chip up and ask for the x,y and z axis of the accelerometer.

X axis function.

This function gives back the x axis integer that the mpu6050 gives back. It does this by reading out the correct addressess and changing those bytes back to an integer by pushing the first byte to the left so that there is an 16 bit number.
